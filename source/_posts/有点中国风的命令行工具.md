`byobu`的图标像是中国的风屏，颇有中国风。哈哈！

官方网址：

[https://www.byobu.org/](https://www.byobu.org/)



**`Ubuntu`下安装`byobu`**

```bash
sudo apt-get install byobu
```

<!--more-->

**运行`byobu`**

安装好后在命令窗口中输入`byobu`即可。



**Byobu常用快捷键**

- F2：新窗口
- F3：前一个窗口
- F4：后一个窗口
- Ctrl + F2：垂直拆分窗口
- Shift + F2：水平拆分窗口
- Shift + 左箭头 / Shift + 右箭头：窗口间导航
